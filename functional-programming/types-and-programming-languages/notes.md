<h1>Definitions</h1>

- <strong class="fancy-title">***Parametric polymorphism***</strong> - allows a single term to be used with many different types.

- <strong class="fancy-title">***Type systems***</strong> - are a way of specifying the types of values in a programming language.

- <strong class="fancy-title">***Subtyping/object types***</strong> - Address the special needs of object-oriented prorgamming styles.

- <strong class="fancy-title">***Connections***</strong> - are being developed between the type systems of programming languages, the specification languages used in program verification, and the formal logics used in theorem proving.

- <strong class="fancy-title">***Term***</strong> - refers to a syntactic piece of a language's code, such as a variable, function, or expression.
